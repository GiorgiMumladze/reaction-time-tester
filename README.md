💬Reaction Time Tester APP
Built with React and Vite
Fully responsive design for all screen sizes
Uses localStorage to save results and history
Tracks average, best, and detailed attempt stats
Interactive click-based gameplay with color feedback
Clean and modern UI with icons and visual indicators
